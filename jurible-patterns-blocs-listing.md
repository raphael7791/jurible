## UTILISATION DANS L'ÉDITEUR

### Insérer un pattern
1. Dans l'éditeur, cliquer sur `+`
2. Onglet "Compositions" (Patterns)
3. Chercher par nom ou catégorie

### Insérer un bloc custom
1. Dans l'éditeur, cliquer sur `+`
2. Chercher "Jurible" ou le nom du bloc
3. Les blocs Jurible sont dans la catégorie "Jurible"

### Appliquer un style natif modifié
1. Sélectionner le bloc (bouton, paragraphe, image...)
2. Dans la sidebar droite, section "Styles"
3. Choisir le style souhaité
